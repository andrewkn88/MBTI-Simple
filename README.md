간단 MBTI 진단 앱
소개
이 앱은 웹브라우저에서 실행되는 간단 MBTI 자기진단 프로그램입니다.
복잡한 설치 없이 `mbti_simple_app.html` 파일만 실행하면 사용할 수
있습니다.
---
주요 기능
16개의 간단한 질문으로 MBTI 성향 진단
MBTI 16가지 유형 자동 판별
유형별 설명 제공
장점 및 강점 소개
단점 및 주의할 점 소개
잘 맞는 활동 분야 안내
대표적으로 알려진 유명인 예시 제공
E/I, S/N, T/F, J/P 점수 표시
결과를 TXT 파일로 저장
---
사용 방법
`mbti_simple_app.html` 파일을 실행합니다.
16개의 질문에 모두 답합니다.
결과 보기 버튼을 누릅니다.
자신의 MBTI 유형과 상세 설명을 확인합니다.
결과 파일 저장하기 버튼을 눌러 TXT 파일로 저장합니다.
