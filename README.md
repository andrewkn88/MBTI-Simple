# 30문항 MBTI 성격 분석 앱

## 소개
웹브라우저에서 실행되는 30문항 MBTI 성격 분석 앱입니다. 참조 사이트처럼 질문에 답하고 결과를 확인하는 흐름을 적용했지만, 문항은 자체 제작했습니다.

## 주요 기능
- 30개 문항
- 5점 척도 응답
- MBTI 16가지 유형 자동 판별
- E/I, S/N, T/F, J/P 점수 표시
- 유형별 설명, 장점, 단점, 성장 조언, 활동 분야 제공
- 대표적으로 자주 언급되는 유명인 예시 제공
- 결과 TXT 파일 저장

## 사용 방법
1. `index.html`을 더블클릭합니다.
2. 30개 문항에 모두 답합니다.
3. **결과 보기**를 누릅니다.
4. 결과를 확인하고 **결과 파일 저장하기**를 누릅니다.

## 파일 구성
```text
MBTI_30_App/
├── index.html
└── README.md
```

|---|---:|

| J / P | 7 |

## 주의사항
이 앱은 자기이해와 교육용 실습을 위한 간단 도구이며 전문 심리검사를 대체하지 않습니다. 유명인의 MBTI는 공식 확인이 아닌 대중적으로 자주 언급되는 추정이 포함될 수 있습니다.
